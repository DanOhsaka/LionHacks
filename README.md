# LionHacks-1

Monorepo for hackathon work. The main application lives in **`learnpulse/`**.

## PridePath (`learnpulse/`)

**PridePath** is a Next.js study companion: upload course material, play checkpoint-based quiz modes, track sessions and analytics, earn achievements, and use wellness check-ins with an in-app coach.

See **[learnpulse/README.md](./learnpulse/README.md)** for stack, environment variables, scripts, and how to run the app locally.
